MaterialInterfaceNeumannBC

buildconstruction:Undocumented Class

The MaterialInterfaceNeumannBC has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Applies a Neumann BC on both sides of the interface. The value of the Neumann BC is specified by a material property.
