CNSFVHLLCMomentumSpecifiedPressureBC

Implements an HLLC boundary condition for the momentum conservation equation in which the pressure is specified.

Overview

This object inherits from CNSFVHLLCMomentumImplicitBC and adds the specified_pressure_function parameter. Evaluation of this provided function is substituted for the implicit pressure value (e.g. the interior pressure value) in boundary flux evaluation.
