Mortar Gap Heat Transfer Action

Description

The MortarGapHeatTransferAction action sets up a set of models used to enforce heat transfer across two surfaces using a mortar approach. These models are separately implemented in user objects. See the description, example use, and parameters on the MortarGapHeatTransfer system documentation page.
