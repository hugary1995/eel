RayTracingNemesis

Outputs ray segments and data as segments using the Nemesis format.

For more information, see RayTracingMeshOutput.

Input Parameters

  • studyThe RayTracingStudy to get the segments from

    C++ Type:UserObjectName

    Controllable:No

    Description:The RayTracingStudy to get the segments from

Required Parameters

  • output_aux_dataFalseWhether or not to also output the Ray's aux data

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Whether or not to also output the Ray's aux data

  • output_dataFalseWhether or not to also output the Ray's data

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Whether or not to also output the Ray's data

  • output_data_nodalFalseWhether or not to output the Ray's data in a nodal sense, in which the data is interpolated linearly across segments

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Whether or not to output the Ray's data in a nodal sense, in which the data is interpolated linearly across segments

  • output_propertiesray_id intersectionsWhich Ray properties to output

    Default:ray_id intersections

    C++ Type:MultiMooseEnum

    Options:ray_id, intersections, pid, processor_crossings, trajectory_changes

    Controllable:No

    Description:Which Ray properties to output

  • use_displacedFalseEnable/disable the use of the displaced mesh for outputting

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Enable/disable the use of the displaced mesh for outputting
